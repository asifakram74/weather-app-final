No, Lagom does not save data from parents' devices. It only keeps a record of your kids' data that you enter in the Lagom app and save it, so you don't have to enter their details again and again.

No, Lagom does not charge you to install the parent app. All you need is to register a parent account to start using the Lagom parents dashboard. You only pay as per the child devices you pair with the parent app.
